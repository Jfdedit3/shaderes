# Z4trox Shaders

Version modifiee de Z4trox Shaders avec une interface retravaillee et un chargement plus prudent.

## Changements

- Theme moderne plus lisible avec surfaces nettes et accents cyan/or.
- Police Gotham, textes plus contrastes, bordures et coins uniformises.
- Boutons, champs texte, barre superieure et scrollbars restyles.
- Chargement distant encadre avec verification d'URL et gestion d'erreurs.
- Envoi de feedback obfusque desactive, car `src/feed.lua` n'est pas auditable proprement.

## Notes

Le fichier `back.json` est du Lua Roblox, pas du JSON strict. A tester dans l'environnement Roblox cible avant usage.
